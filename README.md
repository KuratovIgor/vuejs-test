# Vuejs test

## Start app via Docker
```
docker-compose up
```
And [open in browser](http://localhost:3000)

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```

### Run tests
```
yarn test
```
